# snake-game
Summary: this is a fun learning java based snake game project making use of Java Swing and AWT libraries

- GUI Development: Built a graphical user interface using Java Swing and AWT, utilizing JFrame for the window structure and JPanel for the game canvas.
- Object-Oriented Design: Applied OOP principles such as inheritance (extending JPanel) and encapsulation. Implemented a custom inner class to handle specific keyboard events via the KeyAdapter class.
- Game Logic & Algorithms: Engineered the core game engine using arrays to track coordinate movements and implemented collision detection algorithms to handle boundary and self-intersection rules.
- Event Handling: Utilized the ActionListener interface and Timer class to manage the runtime loop and dynamic rendering of graphics
